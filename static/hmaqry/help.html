About

This web application can be used to query the TxDOT SiteManager database for hot-mix asphalt (HMA) related items.

Query

Queries are run based on user defined settings. The items to include in the query are selected in the left panels. This allows queries to be tailored to specific mixtures, locations, etc, based on the items selected in each of the panels. These panels include multi-selection parent/child type trees to facilitate the selection or de-selection of query groups as defined below. Selecting the "Root" node will select all of its children. Once all query selections have been made, the Data Query panel provides options to extract information from the SiteManager database.

Mixtures

Select the HMA mixtures to query by clicking the respective checkboxes. HMA mixtures have been sub-divided by diffferent mixture types:

  • Dense-graded asphalt (DGA)
  • Permeable friction course (PFC)
  • Performance-related mixtures (PERF)
  • Stone-matrix asphalt (SMA)
  • Thin overlay mixtures (TOM)
  • Thin bonded wearing course (TBWC)
  • Crack attenuating mixtures (CMA)
This grouping provides an initial breakdown of mixtures listed in SiteManager. Within each of these groups, further subdivision by type is possible. Thus PERF mixtures, for example, include both Superpave (SP) and coarse matrix high binder (CMHB) mixtures, etc. Selecting the parent node will select each of its associated children.

All mixtures are selected by default.

Locations

A breakdown of districts and counties within these districts is provided in the Texas tree to restrict queries to specific locations if required.

All locations are selected by default.

From/To Date

Each of these options allows restricting the query to extract data between the selected from and to dates. If only a from date is provided, then the query will extract all records from that specific date. If a to date is provided, all records up until the to date will be extracted. If neither a from nor a to date is provided, the query will extract records regardless of date. The date query uses the date that a lot was completed to identify these records.

No from and to dates are selected by default.

Data Query

Select the type and nature of the data to query:

Agency

The query processes and outputs either TxDOT (default) or contractor data.

Level

Indicate how the data should be summarized, either by project (default) or by lot. If by project, data collected on all lots within the project will be averaged. If by lot, data collected on all sublots will be averaged.

Select query

Select the specific data to query. The follwing options are provided:

  • Lab Density: The density of laboratory comapcted specimen.
  • Voids in the mix (VIM): The air voids in the mix determined from road cores.
  • Voids in the mineral aggregate (VMA): VMA determine from plant mix.
